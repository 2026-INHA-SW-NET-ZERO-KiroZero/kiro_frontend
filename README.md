# KiroZero

Expo + TypeScript 기반 React Native 앱.

## 실행

```bash
npm install        # 의존성 설치
npm start          # Expo 개발 서버 시작 (Expo Go 앱으로 QR 스캔)
npm run ios        # iOS 시뮬레이터
npm run android    # Android 에뮬레이터
npm run web        # 웹 브라우저
```

## 스택

- Expo ~56
- React Native 0.85
- React 19
- TypeScript (strict)

## 구조

- `App.tsx` — 앱 진입 컴포넌트
- `index.ts` — 엔트리포인트
- `assets/` — 아이콘 / 스플래시 이미지
- `app.json` — Expo 설정
