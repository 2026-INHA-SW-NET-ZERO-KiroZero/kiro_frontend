## 목표

`useHomeRooms()` · `useRooms()` · `useIngredients()` 훅 내부를 더미 시드 대신 실제 API로 교체한다.
방 상세(roomDetail.tsx)에서 slotId 라우터 파라미터를 읽고, join/leave API를 연결한다.
화면 shape(PRD §5)는 그대로 유지한다.

## 타입 매핑 노트

### SlotListItemResponse → Room

| API 필드                   | Room 필드     | 변환                                                                       |
| -------------------------- | ------------- | -------------------------------------------------------------------------- |
| `slotId` (long)            | `id` (string) | `String(slotId)`                                                           |
| `placeName`                | `title`       | `placeName` 그대로                                                         |
| `date` (ISO: '2026-06-27') | `date`        | `formatSlotDate(date)` → '06.27 (토)'                                      |
| `startTime`                | `time`        | 그대로 ('16:00')                                                           |
| `placeName`                | `place`       | 그대로                                                                     |
| —                          | `tags`        | 고정 `['저녁', '조리실습실']` (API 미제공)                                 |
| `capacity`                 | `capacity`    | 그대로                                                                     |
| `participantCount`         | `baseCount`   | 그대로                                                                     |
| —                          | `host`        | 빈 문자열 (API 미제공)                                                     |
| —                          | `parts`       | 목록: 빈 배열; 상세: participants에서 매핑, 색은 `avatarPalette` 랜덤 선택 |
| `commonKitSummary`         | `prev`        | 빈 배열 (이모지 아닌 텍스트라 화면 제외)                                   |

### SlotDetailResponse → Room (상세 화면용)

- `participants` → `parts`: `nickname.charAt(0)` + `avatarPalette[i]` 색 매핑
- `joined` → roomDetail의 `joined` 초기 상태
- `myParticipantId` → 방 나가기 등에서 활용

### IngredientItemResponse → Ingredient

| API 필드        | Ingredient 필드                      |
| --------------- | ------------------------------------ |
| `ingredientId`  | `id`                                 |
| `nameKo`        | `nameKo`                             |
| `gramsPerCount` | `gramsPerCount`                      |
| —               | `allergenTags` (optional, undefined) |
| —               | `carbonFactor` (optional, undefined) |
