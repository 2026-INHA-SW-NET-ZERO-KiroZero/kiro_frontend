## 폰트 family 전략 (RN 제약 반영)

RN 커스텀 폰트는 `fontWeight`로 cut을 선택하지 못한다(iOS/Android 모두 신뢰 불가). 따라서 **weight별 family name**을 `expo-font`에 등록하고, 컴포넌트는 `fontFamily`로 weight를 고른다.

- 로드 weight: SemiBold(600) / Bold(700) / ExtraBold(800). PRD "600 미만 사용 안 함" → 3개만.
- `theme.font.family`(신규)를 weight 키와 동일하게 정렬:
  - `family.medium` = `Pretendard-SemiBold` (600)
  - `family.semibold` = `Pretendard-Bold` (700)
  - `family.bold` = `Pretendard-ExtraBold` (800)
  - `font.weight`(기존 numeric)는 참조용으로 유지.
- 기존 `fontWeight: font.weight.X` 사용처 → `fontFamily: font.family.X`로 마이그레이션(동일 weight 유지).
